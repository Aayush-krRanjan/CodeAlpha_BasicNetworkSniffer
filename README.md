# Basic Network Sniffer

## 📌 Project Overview

This project was developed as part of the CodeAlpha Cyber Security Internship.

The Basic Network Sniffer is a Python-based tool that captures live network packets and displays useful information such as the source IP address, destination IP address, protocol type, source port, destination port, and packet payload (if available).

The project helps beginners understand how data travels across a network and how different protocols communicate with each other.

---

## 🎯 Objective

The objective of this project is to:

- Capture live network traffic.
- Analyze packets in real time.
- Display network information.
- Learn how TCP/IP communication works.
- Understand packet structures using Scapy.

---

## 🛠 Technologies Used

- Python 3
- Scapy
- Kali Linux
- VS Code

---

## 📂 Features

✔ Capture live packets

✔ Detect TCP, UDP and ICMP packets

✔ Display Source IP

✔ Display Destination IP

✔ Display Source Port

✔ Display Destination Port

✔ Display Payload (if available)

✔ Continuous packet monitoring

---

## 🚀 How to Run

Install Scapy

pip install scapy

Run the program

sudo python3 sniffer.py

Generate network traffic by opening websites or using:

ping google.com

---

## 📸 Output

The program displays:

- Source IP Address
- Destination IP Address
- Protocol
- Port Numbers
- Packet Payload
